
## Oracle SQL Sales Analytics Project

## Contents

-DDL (tables, primary keys, foreign keys)
-DML (sample data insertion)
-PL/SQL (basic procedural logic)
-Reporting SQL (business reports using JOIN, GROUP BY, CASE)
-Basic analytics (customer and product performance)


## How to Run

-Execute DDL scripts to create tables and constraints.
-Run DML scripts to insert sample customer, product, and sales data.
-Execute PL/SQL scripts to apply procedural logic and calculations.
-Run reporting SQL queries to generate business reports.

## Database Tables

-CUSTOMERS - stores customer details
-PRODUCTS - stores product information
-SALES - stores sales transactions
